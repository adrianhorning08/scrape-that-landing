Frequently Asked Questions

What does this scrape?
+
Only text that appears on the page. If the data is not on the page you submit, we can't scrape it, so far.
What sites does this work for?
+
It works with most sites, but won't work with sites that require logins, or well secured sites. For example, it won't work with Facebook or Instagram.
Is it legal to scrape?
+
Yes. Scraping publicly available data is perfectly legal. We don't scrape behind logins.
Does this work with No Code tools like Zapier and Make?
+
Yes! Just make an API call. You do have to be slightly technical to set it up. Soon there will be a dedicated integration.
